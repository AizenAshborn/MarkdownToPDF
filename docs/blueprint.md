# **App Name**: Markdwn2PDF

## Core Features:

- Real-time Markdown Preview: Display a live, automatically updating preview of the converted HTML content as the user edits the Markdown.
- PDF Conversion and Export: Convert Markdown content into a PDF document using jsPDF.
- Template Selection: Choose from a selection of pre-designed PDF templates to style the output.
- Custom PDF Styling: Customize PDF output with options for fonts, colors, margins, headers, and footers, with a tool for intelligently incorporating all desired parameters.
- Statistics Dashboard: Conversion Statistics for Template Usage using charts

## Style Guidelines:

- Primary color: Deep charcoal (#2D3748) for a professional feel.
- Background color: Desaturated charcoal (#F1F3F5), light scheme.
- Accent color: Amber (#F6AD55) for interactive elements.
- Display Font: 'Playfair', serif, for headings.
- Body Font: 'Inter', sans-serif, for content.
- Code font: 'JetBrains Mono' for markdown/code blocks. Note: currently only Google Fonts are supported.
- Clean and spacious layouts, modern editorial aesthetic inspired by premium publishing platforms, generous white space, subtle shadows and depth for visual hierarchy
- Smooth and purposeful animations and transitions, with gradient text animations, hover transitions, and subtle particle system in the background.
